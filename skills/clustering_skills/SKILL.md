# Unsupervised Clustering Analysis Skill

## Description
Group similar data points together using various clustering algorithms with automatic optimal cluster detection.

## Capabilities
- K-Means clustering
- DBSCAN (density-based)
- Hierarchical clustering
- Gaussian Mixture Models
- Optimal cluster number detection
- Silhouette score analysis
- PCA visualization
- Cluster profiling and interpretation

## Instructions
1. Scale features appropriately
2. Detect optimal number of clusters using elbow method and silhouette scores
3. Apply multiple clustering algorithms
4. Evaluate using silhouette, Davies-Bouldin, Calinski-Harabasz scores
5. Visualize clusters using PCA/t-SNE
6. Profile each cluster with statistics
7. Generate cluster labels for new data

## Example Use Cases
- Customer segmentation
- Image segmentation
- Document clustering
- Anomaly detection
